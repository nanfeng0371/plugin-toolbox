调课助手是什么

调课助手是一个帮你批量改约课程时间的工具。以前你需要在调课后台一个一个点、一个一个改，现在只需要把要改的信息输进去，一键就能全部搞定。

输入调课信息
→
一键解析
→
自动批量改约
💡 举个例子
王一的第5讲要调到5月2日早上10点，你只需要输入：
王一，第5讲，调到5月2日早上10点
工具会自动帮你完成改约，不用再去后台手动操作。
2

使用前的准备

你需要两样东西：

💻 电脑（获取 Token 用）

需要已经安装了调课助手浏览器扩展

需要已经登录了调课后台

（每天只需操作一次）

📱 手机（实际调课用）

打开调课助手网页即可

可以用微信/企微里的链接直接打开

（不需要安装任何东西）

⚠️ 注意
Token 相当于你的登录凭证，有效期 24 小时。过期后需要重新从电脑端复制一次。建议每天上班第一件事先复制一下，当天就不用担心过期了。
3

第一步：获取 Token（电脑端操作）

Token 是一串以 eyJ 开头的长字符串，相当于你的"登录凭证"。有了它，手机端才能代替你在后台操作。

操作步骤

  1. 在电脑浏览器上，打开调课后台网站（就是平时上课管理的那个网站），确保你已经登录了
  2. 点击浏览器右上角的调课助手扩展图标（📚 图标）
  3. 弹出窗口中，点击 🔑 Token 标签页
  4. 如果显示 ✅ Token 有效，点击下方的 📋 复制 Token 按钮
  5. 打开企微或微信，把 Token 发给自己（发到文件传输助手也行）
✅ 成功标志
按钮变成 ✅ 已复制！ 就说明复制成功了。
⚠️ 如果显示"Token 已过期"
需要先重新登录调课后台网站，登录后再回到扩展点🔄 刷新检测，看到 Token 有效后再复制。
🔒 安全提醒
Token 就是你的登录凭证，不要发给别人！只发给你自己的企微/微信。用完之后聊天记录里的 Token 也可以删掉。
4

第二步：粘贴 Token 登录（手机端操作）

拿到 Token 后，在手机上完成登录。

操作步骤

  1. 在手机上打开调课助手网页（从企微/微信里的链接进入，或者输入网址）
  2. 页面会显示 🎯 调课助手 — 粘贴 Token 即可使用
  3. 在输入框里长按 → 粘贴你刚复制的 Token
  4. 粘贴后，页面会自动验证并跳转到调课页面
✅ 成功标志
页面跳转后，顶部显示 🔑 Token：✅ 就说明登录成功了。
⚠️ 如果显示"Token 格式不正确"
说明粘贴的内容不完整。Token 很长，确保整串都复制了，以 eyJ 开头，中间有两个 . 分隔。回到企微/微信，长按消息全选复制，再粘贴一次。
⚠️ 如果 Token 过期了
顶部显示 🔑 Token：❌ 已过期，需要回到第3步重新从电脑复制。
💡 另一种粘贴方式
如果你的手机浏览器没自动跳转，也可以：
1. 点击底部 「设置」 标签
2. 找到 🔑 调课 Token 区域
3. 点击 📋 粘贴 按钮或手动粘贴到输入框
4. 点击 ✅ 验证并保存
5

第三步：输入调课指令

登录成功后，在 「调课」 标签页的输入框中，输入你要调课的信息。支持两种写法，你习惯哪种就用哪种：

写法一：自然语言（推荐，最简单）

就像平时说话一样写，每行写一条：

📝 输入示例
王一，第5讲，调到5月2日早上10点上课
李二，第3讲，改到6月5号下午2点半
张三，第8讲，调到5月10号晚上7点
💡 自然语言的写法很灵活
关键词："第X讲" 必须有，日期和时间也必须有
连接词："调到" "改到" "约到" 都可以
时间段："早上" "上午" "下午" "晚上" 都支持
时间写法："10点" "10:00" "2点半" "14:30" 都行
日期写法："5月2日" "5月2号" "5-2" "2026-05-02" 都行

写法二：结构化（适合批量复制）

每行四个信息，用空格或Tab隔开：

📝 输入示例（空格分隔）
862719  11  5-27  10:00
865432  5   6-1   14:00
861234  3   5-30  9:00

四个字段的意思是：

第1列 第2列 第3列 第4列
学员ID 或 姓名 第几讲（数字） 新日期 新开始时间
862719 或 王一 11 5-27 10:00
💡 用姓名代替学员ID
如果你不记得学员ID，也可以直接输入姓名。但前提是先在「设置」页导入学情表（Excel文件），这样工具才能把姓名对应到学员ID。

如果没导入学情表，就只能用学员ID（纯数字）。
6

第四步：解析与执行

6.1 先解析预览

  1. 输入调课指令后，点击 📋 解析预览 按钮
  2. 页面会显示解析结果，告诉你哪些能识别、哪些有问题
  3. 检查一下，确认无误后进入下一步
✅ 解析成功示例
显示类似：共 3 条：✅ 3 条成功，❌ 0 条失败
每条下面显示：学员 → 第X讲 → 新日期 新时间
⚠️ 如果有解析失败的
比如显示 ❌ 未找到学员"某某"，说明姓名在学情表里没有匹配到。可以：
1. 改用学员ID（纯数字）输入
2. 检查姓名是否写错了

6.2 执行调课

  1. 确认解析结果无误后，点击 ▶ 执行调课 按钮
  2. 等待执行完成（一般几秒钟就搞定了）
  3. 查看结果：✅ 成功 的表示调课已完成，❌ 失败 的会说明原因
✅ 全部成功
显示类似：共 3 条：✅ 3 条成功，❌ 0 条失败
调课完成！可以去调课后台刷新页面确认。
⚠️ 部分失败怎么办
1. 查看失败原因（比如"未找到第X讲的课表记录"）
2. 确认学员ID和第几讲是否正确
3. 修正后重新输入，再次执行

输入格式详解

7.1 日期支持的写法

写法 示例 说明
中文日期 5月2日 5月2号 最常用，简单直观
短横线 5-2 05-02 省略年份，自动用今年
斜杠 5/2 05/02 省略年份，自动用今年
完整日期 2026-05-02 带年份的写法

7.2 时间支持的写法

写法 示例 说明
标准时间 10:00 14:30 最常用
中文"点" 10点 下午2点 支持时间段词自动转换
"点半" 2点半 下午2点半 自动转为 :30

7.3 时间段自动转换

💡 不用纠结12小时制还是24小时制
当你写 下午2点 时，工具会自动转成 14:00
当你写 晚上7点 时，工具会自动转成 19:00
当你写 上午10点 时，就是 10:00，不会变
如果你直接写 14:00，那工具直接用，不需要转换

7.4 课程时长

💡 结束时间自动计算
课程时长固定为 2 小时。你只需要输入开始时间，结束时间会自动计算。
例如：开始时间 10:00 → 结束时间 12:00

7.5 更多输入示例

📝 各种写法都行
862719  11  5月27日  10:00         ← 日期用中文写法
862719  11  5-27  10:00            ← 日期用短横线
862719  11  2026-05-27  10:00      ← 日期带年份
王一，第11讲，调到5月27日10点      ← 自然语言+姓名
862719  11  5-27  下午2点          ← 时间用中文
把张三的第3讲改到6月1号上午9点      ← 纯自然语言

常见问题（FAQ）

Q：Token 是什么？我需要自己创建吗？
Token 是你登录调课后台后自动生成的凭证，不需要自己创建。你只需要在电脑上调课助手扩展里点「复制 Token」就行。
Q：Token 多久过期？
24 小时。过期后需要重新从电脑端复制一次。建议每天上班第一件事先复制，当天就不用担心。
Q：我没有电脑，只有手机怎么办？
目前获取 Token 需要电脑端的浏览器扩展。可以让同事帮忙复制 Token 发给你，你在手机上粘贴即可。但请注意，Token 不要发给不相关的人。
Q：输入姓名为什么提示"未找到学员"？
需要先在「设置」页导入学情表（Excel文件），工具才能把姓名匹配到学员ID。如果没导入学情表，请直接用学员ID（纯数字）。
Q：学情表是什么格式？
Excel 文件（.xlsx 或 .xls），Sheet1 的第一行是表头，需要包含"姓名""手机号""学员ID"三列（列名支持多种写法，工具会自动识别）。
Q：调课失败了怎么办？
1. 先看失败原因（会显示在结果里）；2. 确认学员ID和第几讲是否正确；3. 确认 Token 是否过期；4. 修正后重新输入执行。
Q："第几讲"填错了会怎样？
工具会找不到对应的课表记录，调课会失败。请确认你要调的是第几讲（在调课后台可以看到）。
Q：一次最多能调多少条？
没有硬性上限，但建议一次不超过 20 条。工具会自动3条一组并发执行，速度很快。
Q：调课操作可以撤销吗？
调课助手本身没有撤销功能。如果调错了，需要手动在调课后台改回来，或者用调课助手再调一次。建议执行前先仔细检查解析结果。
Q：手机上打开网页一直加载/白屏？
1. 检查网络是否正常；2. 试试刷新页面；3. 如果问题持续，可能是 Token 过期了，重新从电脑端复制。
Q：我把 Token 发给别人了怎么办？
Token 相当于你的登录凭证，别人拿到后可以用你的身份操作。建议：1. 尽快在调课后台退出登录重新登录，旧 Token 会自动失效；2. 删除聊天记录中的 Token。
