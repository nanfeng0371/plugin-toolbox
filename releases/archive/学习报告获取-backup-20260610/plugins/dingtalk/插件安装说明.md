# 钉钉数据提取助手 - Edge浏览器插件

## 🎉 终于简单了！

这是一个**Edge浏览器插件**，一键提取钉钉网页表格数据，无需任何代码操作！

---

## ✨ 功能特点

- ✅ **一键启动** - 点击插件图标即可开始
- ✅ **自动滚动** - 智能识别并滚动页面
- ✅ **智能去重** - 自动识别重复记录
- ✅ **自动下载** - 数据自动保存为CSV文件
- ✅ **可视化界面** - 美观的控制面板
- ✅ **可调速度** - 自定义滚动延迟

### 方法1：开发者模式安装（推荐）

1. **打开Edge浏览器**
2. **输入地址**：`edge://extensions/` 按 Enter
3. **启用"开发者模式"**（右上角开关）
4. **点击"加载已解压的扩展"**按钮
5. **选择本文件夹**：
   ```
   c:/Users/wy260/WorkBuddy/20260316171000
   ```
6. **完成！** 插件图标会出现在浏览器右上角

### 方法2：创建图标文件（如果提示缺少图标）

插件需要图标文件，请按以下步骤创建：

#### 方法A：使用在线工具
1. 访问 https://www.favicon-generator.org/

1. 创建3个PNG图片：
   - 16x16 像素
   - 48x48 像素
   - 128x128 像素
2. 使用蓝色背景（#1890ff）
3. 命名为 `icon16.png`, `icon48.png`, `icon128.png`
4. 保存到插件文件夹

## 🚀 使用方法

### 第一次使用

1. **打开钉钉网页**并登录
2. **导航到数据页面**
3. **点击插件图标**（浏览器右上角的钉钉数据提取助手）
4. **点击"显示控制面板"**
5. 页面右上角会出现控制面板

### 开始提取

1. 在控制面板上点击 **"▶ 开始提取"** 按钮
2. 等待自动滚动提取（约5-10分钟）
3. 数据会自动下载为CSV文件到电脑

### 高级功能

- **调整速度**：拖动滚动延迟滑块（500ms-3000ms）
- **停止提取**：点击"⏹ 停止"按钮
- **手动下载**：点击"💾 下载CSV"按钮

## ❓ 常见问题

### Q1: 安装时提示"缺少manifest.json"？
**解决**：确保选择了正确的文件夹，包含 `manifest.json` 文件

### Q2: 安装时提示"缺少图标文件"？
**解决**：按照上面的"创建图标文件"方法创建图标

### Q3: 点击插件图标没有反应？
**解决**：
1. 刷新页面
2. 确保在钉钉网页上操作
3. 检查控制台是否有错误信息

### Q4: 提取到的数据不完整？
**解决**：
1. 调整滚动延迟（增加延迟时间）
2. 等待表格完全加载后再开始提取
3. 检查是否有分页或懒加载

### Q5: CSV文件打开乱码？
**原因**：Excel/WPS默认编码与CSV文件编码不匹配

**解决方法**：

**方法A（推荐）：更新插件后重新提取**
1. 打开 `edge://extensions/`
2. 点击"钉钉数据提取助手"的刷新按钮 🔄
3. 重新提取数据并下载

**方法B：手动设置编码（Excel）**
1. 右键点击CSV文件 → 打开方式 → 选择 Excel
2. 在"文件 → 打开"时，选择"导入"
3. 导入向导中，选择原始数据类型：Unicode (UTF-8)

**方法C：手动设置编码（WPS）**
1. 用WPS打开CSV文件
2. 如果显示乱码，关闭文件
3. 重新打开时，在编码选择中选择：UTF-8

**方法D：使用Excel导入功能**
1. Excel中：数据 → 获取数据 → 从文本/CSV
2. 选择CSV文件
3. 文件原始格式选择：65001: Unicode (UTF-8)
4. 分隔符选择：逗号
5. 点击"加载"

### Q6: 如何卸载插件？
**解决**：
1. 访问 `edge://extensions/`
2. 找到"钉钉数据提取助手"
3. 点击"删除"按钮

## 💡 使用技巧

### 1. 提高提取效率
- 调整浏览器窗口大小，让表格显示更多行
- 使用较快的滚动延迟（800-1000ms）
- 确保网络稳定，避免数据加载失败

### 2. 数据去重原理
- 使用表格的第一列（序号）或姓名+电话作为唯一标识
- 自动跳过已提取的记录
- 确保数据不重复

### 3. 处理大量数据
- 如果数据超过1000行，建议分批次提取
- 每次提取后下载备份
- 最后合并所有CSV文件
