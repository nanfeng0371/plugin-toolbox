## 🆕 新增功能

### 1. 🗓️ 课程排期分析看板（全新模块）

**功能入口**：侧边栏 → 🗓️ 课程排期分析 Tab

#### 核心能力
- **仪表盘视图**：统计卡片（总排课/过载学员/异常学员/闲置学员）+ 过载/闲置 Top5 列表 + 异常检测面板
- **全屏热力图**：点击「查看完整热力图」打开独立页面，28列×7行完整热力图，支持鼠标悬停查看详情、点击高亮
- **排课明细面板**：点击日期格子 → 按学员下钻，5列精简表格（姓名/ID/讲次/时段/在线），支持 CSV 导出
- **导出日历图片**：Canvas 渲染排课详情表 + 统计信息，支持下载 PNG 图片

#### 异常检测（自动执行）
| 异常类型 | 说明 |
|---------|------|
| 🚨 日期越界 | 排课日期超出合理范围（如暑假课外的时间）|
| ⚠️ 连续排课 | 同一学员连续多天排课（建议间隔休息）|
| 📊 每日上限 | 单日排课超过设定阈值（默认1节/天）|
| 🔍 数据异常 | API 返回数据字段缺失或格式异常 |

### 2. 📦 分析结果持久化（学习报告模块）

**问题修复**：分析完成后关闭侧边栏或刷新页面，结果丢失，需重新分析。

**新行为**：
- 分析完成自动保存到本地存储
- 重新打开侧边栏/刷新页面 → 自动恢复上次结果
- 状态栏显示：`📦 已恢复上次分析 | 144个学生 | 5分钟前`

## 🔄 升级操作指南

### 方式一：自动更新（推荐）
1. 关闭所有浏览器窗口
2. 重新打开浏览器
3. 扩展会自动检测更新并安装

### 方式二：手动重新加载
1. 打开 `edge://extensions/`
2. 找到「插件工作箱」
3. 点击「重新加载」按钮
4. **必须**：关闭所有浏览器窗口，重新打开（Native Host 更新需要）

### 验证升级成功
- 侧边栏 → 检查右上角版本号应为 `v2.1.57`
- 或打开 `edge://extensions/` → 插件工作箱详情 → 版本 `2.1.57`

---

## ⚠️ 注意事项

1. **Native Host 更新**：如果之前安装过 Native Host，升级后需关闭浏览器重开（否则 Native Host 版本可能不匹配）
2. **热力图模块首次使用**：需打开课堂数据页面（aitutor100 或 next.aitutor100.com）才能使用
3. **报告模块持久化**：恢复的数据保留在本地，清除浏览器数据会同时清除保存的分析结果
