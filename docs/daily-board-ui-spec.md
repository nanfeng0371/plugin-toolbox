## 六、分隔条交互

### 6.1 视觉

```
┌──────────────────────────┬─┬───────────────────────────┐
│  原页面                   │║│  看板面板                  │
│                          │║│                           │
│                          │║│  ← 6px 分隔条              │
│                          │║│    2px 可视手柄（灰色）     │
└──────────────────────────┴─┴───────────────────────────┘
```

- 分隔条宽 6px，其中 2px 为可视手柄（`#d0d5dd`）
- hover 时手柄变蓝（`#2196f3`），cursor 变为 `col-resize`
- mousedown 开始拖拽，mousemove 实时调整面板宽度
- mouseup 结束拖拽，更新 body margin-right
- **最小宽度保护**：面板不小于 420px
- **最大宽度保护**：面板不超过 55vw

### 6.2 拖拽实现伪代码

```javascript
let isDragging = false;
let startX, startWidth;

divider.addEventListener('mousedown', (e) => {
  isDragging = true;
  startX = e.clientX;
  startWidth = panel.offsetWidth;
  document.body.style.userSelect = 'none'; // 拖拽时禁止选中文字
  document.body.style.cursor = 'col-resize';
});

document.addEventListener('mousemove', (e) => {
  if (!isDragging) return;
  const delta = startX - e.clientX; // 向左拖 → 面板变宽
  let newWidth = startWidth + delta;
  // 范围保护
  newWidth = Math.max(420, Math.min(newWidth, window.innerWidth * 0.55));
  panel.style.width = newWidth + 'px';
  document.body.style.marginRight = newWidth + 'px';
});

document.addEventListener('mouseup', () => {
  if (!isDragging) return;
  isDragging = false;
  document.body.style.userSelect = '';
  document.body.style.cursor = '';
  // 保存宽度偏好到 chrome.storage.local
  chrome.storage.local.set({ dailyboard_panel_width: panel.offsetWidth });
});
```

## 八、数据合并逻辑（防止打勾丢失）

### 8.1 问题

刷新时会重新 `classifyStudents()`（生成全新的 `students[]`），如果直接覆盖 → 之前打的勾全丢。

### 8.2 解决方案

```javascript
function mergePrevStates(newStudents, prevStates) {
  // prevStates = { "273719": { isDone: true, doneTime: "...", rebooked: true }, ... }
  return newStudents.map(s => {
    const prev = prevStates[s.studentId];
    if (prev) {
      return { ...s, isDone: prev.isDone, doneTime: prev.doneTime, rebooked: prev.rebooked };
    }
    return s; // 新学生，无历史状态
  });
}
```

### 8.3 学情表重新粘贴的处理

```
重新粘贴学情表
    ↓
保留 prevStates（所有历史打勾状态）
    ↓
重新 classifyStudents + mergePrevStates
    ↓
旧学情表有、新学情表无的学生 → prevStates 中保留（标记为 "已移除"）
新学情表有、旧学情表无的学生 → 正常新增，isDone = false
```

## 十二、技术实现细节

### 12.1 面板 DOM 结构

```html
<div id="db-right-panel" class="db-panel" style="width: 500px;">
  <!-- 分隔条（左边缘） -->
  <div id="db-divider" class="db-divider">
    <div class="db-divider-handle"></div>
  </div>
  
  <!-- A 区：顶部固定 -->
  <div class="db-header-fixed">
    <div class="db-titlebar">...</div>
    <div class="db-infobar">...</div>
    <div class="db-toolbar">...</div>
  </div>
  
  <!-- B 区：可滚动内容 -->
  <div class="db-scroll-area">
    <div class="db-categories">...</div>
  </div>
  
  <!-- C 区：底部状态栏 -->
  <div class="db-footer-fixed">...</div>
</div>
```

### 12.2 注入位置

```javascript
// 直接注入到页面 body（非 Shadow DOM），避规避 view transform 问题
const panel = document.createElement('div');
panel.id = 'db-right-panel';
document.body.appendChild(panel);

// 设置页面推开
document.body.style.marginRight = panelWidth + 'px';
document.body.style.transition = 'margin-right 300ms ease';
```

### 12.3 样式隔离

- 使用 Shadow DOM 包裹面板内部样式
- CSS 类名前缀 `.db-`
- 面板自身 `all: initial` 重置外部样式影响
- 分隔条样式直接作用于面板，无需 prefix

### 12.4 滚动行为

```
面板内 B 区独立滚动:
.db-scroll-area {
  flex: 1;
  overflow-y: auto;
  overflow-x: hidden;
}

原页面正常滚动:
不受面板影响，body 的 overflow 保持不变
```

## 十三、样式规范

| 元素 | 规范 |
|------|------|
| 字体 | -apple-system, BlinkMacSystemFont, sans-serif |
| 面板内边距 | 0（A/B/C 区各自管理自己的 padding） |
| A区内边距 | 12px |
| B区内边距 | 卡片间距 8px，卡片内 12px |
| 行高 | 学生行 36px |
| checkbox | 16×16px，accent-color 跟随分类颜色 |
| 点击区域 | 整行可点击（点击行 = 点击 checkbox） |
| 分隔条 | 6px 宽，2px 可视手柄 |
| 分类颜色 | 1灰/2蓝/3橙/4绿/5蓝/6红/7灰 |
| CSS 前缀 | `.db-` |
| 最小窗口宽度 | 900px（低于此宽度用浮动模式，不推开页面） |
| 动画 | 300ms ease（面板滑入/滑出，body margin 过渡） |

### 窄窗口降级（< 900px）

```
当 window.innerWidth < 900px 时:
  → 面板改为浮动覆盖模式（position: fixed; left: 0; width: 100vw; z-index: 999999）
  → body margin-right 不设置（页面不被推开）
  → 面板有半透明黑色遮罩背景（rgba(0,0,0,0.3)）
  → 面板居中，最大宽度 500px，带圆角阴影
```
